## 🔢 EV Calculator - Try It Yourself!

I built a Python calculator that shows:
- How much money you save with an EV
- Your subsidy eligibility based on income
- India's electricity needs by 2040

### How to Run This Code

**Option 1: Online (Easiest)**
1. Go to [replit.com](https://replit.com)
2. Create new Python file
3. Copy `ev_calculator.py` from this repository
4. Click Run

**Option 2: On Your Computer**
1. Install Python from python.org
2. Download `ev_calculator.py`
3. Open terminal and type: `python ev_calculator.py`
